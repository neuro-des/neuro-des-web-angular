Info
In order to run the simulation, the software requires a generated 'Map Schema' of the area you want to run the simulation. The map contains the following information:
  1. The total catchment area
  2. The location, type and capacity of each hospital
  3. The catchment areas of each hospital
  4. A randomly generated set of possible patient locations, with driving times calculated to their nearest hospital (using google maps algorithms)
In future version, there will be a separate application used to generate this map data that you will be able to upload here.
Currently, you are able to select from a list of pre-built Map Schema that the development team have created. If there is a schema that you would like to see us create, please contact us from the main page.
